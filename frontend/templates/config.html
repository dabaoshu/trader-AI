配置指导
QQ邮箱配置步骤：
  1. 1. 登录QQ邮箱
  2. 2. 设置 → 账户 → 开启SMTP服务
  3. 3. 生成授权码（非QQ密码）
  4. 4. 在上方填入授权码
163邮箱配置步骤：
  1. 1. 登录163邮箱
  2. 2. 设置 → 客户端授权密码
  3. 3. 开启SMTP服务
  4. 4. 设置客户端授权密码
Gmail配置步骤：
  1. 1. 开启两步验证
  2. 2. 生成应用专用密码
  3. 3. 使用应用密码登录

安全提示：授权码仅用于第三方客户端登录，不是您的邮箱登录密码
