# 🖥️ trader-AI Web管理平台完整指南

## 🎯 产品概述

**trader-AI Web管理平台**是一个专业的股票分析与交易决策管理系统，为您的智能交易助手提供可视化的Web界面，让复杂的股票分析变得直观易用。

### 🌟 核心特色
- **🎨 专业界面**: 现代化Web设计，响应式布局
- **📊 实时监控**: 系统状态实时显示，一目了然
- **🎯 智能推荐**: 可视化股票推荐，详细分析数据
- **⚙️ 便捷配置**: 图形化配置管理，告别命令行
- **📈 数据可视**: 多维度数据展示和筛选功能

## 🚀 快速启动

### 方式一：一键启动（推荐）
```bash
python3 start_web.py
```

### 方式二：直接启动
```bash
python3 web_app.py
```

### 🌐 访问地址
启动成功后，在浏览器中访问：**http://localhost:8080**

## 🛠️ 技术架构

### 🏗️ 系统架构
```
前端界面 (HTML5 + Bootstrap + JavaScript)
    ↓
Web框架 (Flask)
    ↓
业务逻辑 (股票分析 + 邮件系统 + 调度器)
    ↓
数据存储 (SQLite + JSON文件)
    ↓
数据源 (BaoStock + AKShare)
```

### 📦 技术栈
- **前端**: HTML5, CSS3, JavaScript, Bootstrap 5
- **后端**: Python Flask, SQLite
- **数据分析**: Pandas, NumPy
- **图表**: Chart.js
- **图标**: Font Awesome
- **样式**: Bootstrap + 自定义CSS

## 🐛 常见问题排查

### 启动问题
**端口占用**：
- 修改端口号或释放5000端口
- macOS用户可关闭AirPlay接收功能

**依赖包缺失**：
```bash
pip3 install flask pandas numpy akshare baostock schedule
```

**权限问题**：
```bash
chmod +x start_web.py
```

### 功能问题
**邮件发送失败**：
- 检查邮箱配置是否正确
- 确认授权码而非登录密码
- 测试网络连接

**数据获取异常**：
- 检查网络连接
- 验证数据源API状态
- 查看系统日志

**页面显示异常**：
- 清除浏览器缓存
- 检查JavaScript控制台错误
- 确认模板文件完整
